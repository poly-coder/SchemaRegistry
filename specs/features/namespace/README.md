# Namespace Feature

This is the readme file of namespace feature

## Introduction

The namespace feature allows users to create and manage namespaces within the schema registry. A namespace is a logical grouping of schemas that helps organize and isolate them for better management and access control.

## Features

- Create Namespace: Users can create new namespaces by providing a unique name and optional description.
- Soft Delete Namespace: Users can mark a namespace as deleted without permanently removing it.
- Delete Namespace: Users can delete an existing namespace.
- List Namespaces: Users can retrieve a list of all existing namespaces.
- Namespace Details: Users can view detailed information about a specific namespace.
- Export Schemas: Users can export all schemas within a namespace for backup or code generation purposes.
